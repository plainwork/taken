# taken

Taken is a tiny menu bar app for appending your clipboard to Taken notebooks.

## Install

Install via Homebrew tap:

```bash
brew tap plainwork/taken-app https://github.com/plainwork/homebrew-taken-app
brew install --cask taken-app
```

Or install directly:

```bash
brew install --cask plainwork/taken-app/taken-app
```

## Use

- Click the Taken icon in the menu bar to select a notebook and append your clipboard.
- Toggle the notebook picker with `control+option+command+N`.
- Append to the default notebook with `control+option+command+T`.
- Quit from the Quit button or press `command+Q`.
